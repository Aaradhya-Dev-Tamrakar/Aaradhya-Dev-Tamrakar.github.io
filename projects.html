SPARK — Two-Layer Fall Detection Wearable

Fully on-device, two-layer fall-detection wearable with an explainable gateway pipeline for eldercare, built entirely from locally-sourced components — zero imports, zero custom PCB. MPU6050 IMU streams through a complementary filter to a sub-5ms Layer 1 threshold gate, then a Layer 2 TFLite Micro 1D CNN (INT8, <100ms) confirms via MQTT to a Raspberry Pi 4B gateway running FastAPI, per-event SHAP attribution, a Streamlit dashboard, Telegram alerts, and auto-generated clinical-format PDF incident reports — end-to-end fall-to-alert target under 2 seconds. Trained on UCI MHEALTH and SisFall, extended with a self-collected Nepal-context IMU dataset. BEI major project, four-person team — demo target March 2027.
